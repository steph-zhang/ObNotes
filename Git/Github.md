# 1. 创建远程仓库
![[Pasted image 20221026193317.png]]

#### 创建远程仓库别名
![[Pasted image 20221026193746.png]]

基本语法：
```
查看当前所有远程仓库地址别名
	git remote -v

创建远程仓库别名
	git remote add 别名 远程地址
```

# 2. 远程仓库操作

#### 2.1 推送本地分支到远程仓库

基本语法：
```
	git push 别名 分支
```

#### 2.2 拉取远程库

基本语法：
```
	git pull 别名 分支
```

#### 2.3 克隆远程库到本地库

基本语法：
```
	git clone 远程地址
```

#### 2. 4 SSH免密登录

基本语法：
```
生成.ssh密钥目录
	ssh -keygen -t rsa -C 用户名	
```
